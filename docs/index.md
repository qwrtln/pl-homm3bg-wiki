# Strona Główna

## :attack: What Is This? { data-toc-label='What Is This?' }

Prosty zbiór kart zawartych w grze Heroes of Might and Magic 3: The Board Game, który zawiera również dodatkowe wyjaśnienia dotyczące działania niektórych kart oraz ich wzajemnych interakcji.

Co jakiś czas chciałem sprawdzić, co robią niektóre karty - czy to podczas obmyślania strategii, czy nawet w trakcie gry, gdy chcę coś wyjaśnić albo znaleźć odniesienie. Ta baza danych ma na celu zapewnienie łatwego dostępu do dowolnej karty w grze, a także wizualnego odniesienia i dodatkowych informacji tam, gdzie jest to konieczne (lub możliwe).


## :defense: Future Plans? { data-toc-label='Future Plans?' }
